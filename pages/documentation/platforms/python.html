Getting started with Haxe for Python

1

Install Haxe

Download and install the Haxe SDK and Runtime.
2

Install a compatible Haxe IDE

You will need one of these to develop your Haxe projects.
  • Visual Studio Code
    • Install the Haxe Extension Pack
  • IntelliJ IDEA
    • Install the Haxe Plugin
  • Other IDEs
3

Create your project

1. Create a new Haxe project in your IDE
2. Create a .hxml to store compiler config for your project
3. Add the following into the file:
-lib haxe-strings -cp src -python PythonOut/main.py -main Main.hx
The configuration above, specifies that you are using the haxe-strings haxelib (totally optional), that your source code is stored in /src, your main Haxe file is src/Main.hx, and that you want Haxe to output the Python script source code into PythonOut/main.py. After code generation, you will need to manually run or debug your Python program using a specialized Python IDE.
4

Install the Python runtime

Haxe requires Python 3.x. Due to security fixes and new features Python 3.6 or later is recommended.

1. Install Python 3.7 or later.
  • Tested workflow on Windows: Install Python 3.7 64-bit for All Users
  • Python installer does not change any Registry values.
  • Python installer adds the installed Python directory to the PATH environment variable.
2. Test your installation by opening a command prompt and typing "python".
  • You should see something like the following snippet.
  • If your console cannot find python, you have an issue with your installation or python is not added into the PATH environment variable.
Python 3.7.2 (tags/v3.7.2:9a3ffc0492, Dec 23 2018, 23:09:28) [MSC v.1916 64 bit (AMD64)] on win32 Type "help", "copyright", "credits" or "license" for more information. >>>
The >>> prompt is from the Python interpreter and shows it is ready to run .py scripts.
5

Develop your project

Using API and libraries

Browse the haxelib for community-developed libraries that you can add to your project.
Browse the Haxe Python API website for the Python-specific API available.
Browse the Haxe API website for the core Haxe platform API that you can use in your project.

6

Debug your project

Running and debugging

After compiling your project with Haxe, you'll need to open a console in the PythonOut directory and run the command python main.py
You can alternatively use a full-fledged Python IDE like the ones given below to run and debug your script easily.

Install a Python IDE

You will need one of these to run and debug Python scripts generated from your Haxe project.
  • PyScripter
  • Visual Studio 2017 Community Edition
  • Visual Studio Code
7

Ask the community

If you need help with anything, visit the vibrant Haxe community and simply ask for help with the details of your project. We have a helpful and active community and you should get your answers in a few hours!
