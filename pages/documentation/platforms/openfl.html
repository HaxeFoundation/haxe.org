Getting started with Haxe for OpenFl

1

Install Haxe

Download and install the Haxe SDK and Runtime.
2

Install a compatible IDE

Recommended IDEs:
  • FlashDevelop
  • Visual Studio Code
    • Install the Haxe Extension Pack
  • IntelliJ IDEA
    • Install the Haxe Plugin
  • Other IDEs
Recommended tools:
  • HxScout - Visual profiler for OpenFl
3

Create your project

1. Create a new blank folder for your project.
2. Open the console in the context of your project folder.
3. Run the following command to create a new Haxe OpenFl project:
openfl create project MyProjectName
4. If you want to use the HxScout profiler add -Dtelemetry to your openfl command line.
4

Develop your project

Project

Refer to this guide to create a basic Haxe OpenFl project.

Libraries

You can use Tinkerbell to boost your development.
You can use Starling if you don't like the OpenFl graphics API.
You can use Away3D or BabylonHx to display GPU-accelerated 3D graphics.
Browse the haxelib for community-developed libraries that you can add to your project.

API

Browse the Haxe API website for the core Haxe platform API that you can use in your project, which includes system I/O, networking and other core functions.
Browse the OpenFl API website for the graphics-related OpenFl API that you can use in your project.

5

Ask the community

If you need help with anything, visit the vibrant Haxe community and simply ask for help with the details of your project. We have a helpful and active community and you should get your answers in a few hours!
